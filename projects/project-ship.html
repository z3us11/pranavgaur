Ship Horde Survival Game

Prototype | Development Started : 2024
Unity 2D Horde Survival Roguelike Project Management

A 2D horde survival game where players pilot a momentum-based ship and survive increasingly challenging waves of enemies. The core gameplay focuses on mastering ship movement, positioning, and upgrade choices while navigating large enemy hordes.

Technical Breakdown

velocity_raptor.exe
Physics Based Movement
  • Implemented momentum-driven movement using Unity's physics system.
  • Movement and turning mechanics are designed to make ship handling a key gameplay skill rather than simple directional input.
clonewar.exe
Enemy Wave System
  • Modular wave configuration system supporting different enemy types, health scaling, spawn timings, and spawn patterns.
  • Progressive difficulty increases through enemy variety and stat adjustments across waves.
hot_fix.diff
Player Upgrade System
  • Level-up system presents players with a choice of three upgrade options.
  • Upgrades modify core gameplay attributes such as:
    • Health
    • Movement Boost
    • Weapon Shooting Capabilities
  • Designed to encourage different playstyles and build strategies during each run.
